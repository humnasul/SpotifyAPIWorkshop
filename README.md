A Web Application accessing Spotify API and Ticketmaster API for top songs, filtering top songs into categories, and recommending upcoming events!   
Utilizes: Node.js, Spotify for Developers, Ticketmaster Developer, REST API, OAuth 2.0, JSON, HTML, CSS
