A Web Application accessing Spotify API and Ticketmaster API for top songs, filtering top songs into categories, and recommending upcoming events!   
Utilizes: Node.js, Spotify for Developers, Ticketmaster Developer, REST API, OAuth 2.0, JSON, HTML, CSS   
Slides with more info available here: https://docs.google.com/presentation/d/1z10gctya0IdPQCAIAEQlAyj3Lyl2CB8dNWZeKxi9i1c/edit?usp=sharing
- workshop_code contains code with blank entries to fill in during a live workshop
- final_code_with_events contains final code for this application ('with_events' refers to Ticketmaster event recommendations at the bottom of the page)
